## 2024-04-03 - Contextual CTAs in Empty States
**Learning:** Always provide contextual call-to-action buttons in empty states that route users directly to the solution path (e.g., Settings), even if it requires passing an authorization gate like a parental gate. This drastically reduces navigation friction.
**Action:** Review all empty states in the app to ensure they have actionable CTAs rather than just informative text instructing the user what to do.
